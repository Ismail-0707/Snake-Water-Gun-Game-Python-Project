# 🐍 Snake, Water, Gun Game (Python Console Project)

This is a simple and fun console-based game built with **Python**, inspired by the classic **Rock, Paper, Scissors** game. In this version, the choices are **Snake**, **Water**, and **Gun**.

The player competes against the computer, and the winner is decided based on predefined game rules.

## 🎮 Game Rules

- Snake drinks water → Snake wins 🐍💧
- Gun shoots snake → Gun wins 🔫🐍
- Water damages gun → Water wins 💧🔫
- Same choice → It's a tie!

## 🧰 Technologies Used

- Python 3
